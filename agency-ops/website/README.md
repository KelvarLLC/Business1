# Website Positioning Brief

## Purpose

This folder is reserved for the public Kelvar LLC automation agency website. The website should become the flagship public entry point for the agency's AI automation ecosystem.

## Positioning

Kelvar builds practical AI automation ecosystems for service businesses that need faster follow-up, cleaner handoffs, and less manual admin work.

## Target Visitors

- Small service business owners.
- Local operators with repetitive admin or follow-up work.
- Teams using email, forms, spreadsheets, CRMs, scheduling tools, or manual handoffs.
- Prospects who want a practical automation package rather than an open-ended technology project.

## Primary Website Goal

Convert qualified visitors into discovery or audit conversations for narrow, sellable automation packages.

## Recommended Navigation

- Home
- Services
- About
- Contact
- Optional later: Case Studies, FAQ, Resources

## Homepage Message

### Hero Positioning

AI automation systems for service businesses that need fewer missed leads, faster follow-up, and cleaner operations.

### Supporting Copy

Kelvar helps small teams replace scattered manual processes with practical AI-powered workflows for intake, follow-up, onboarding, reminders, reporting, and delivery handoffs.

### Primary CTA

Request an automation audit.

### Secondary CTA

View service packages.

## Service Packages To Feature

1. Lead intake and follow-up automation.
2. Client onboarding and handoff automation.
3. Appointment reminder automation.
4. Lightweight reporting dashboard automation.

## Tone Guidelines

- Professional, direct, and practical.
- Outcome-focused rather than hype-heavy.
- Confident about automation value without promising guaranteed revenue.
- Use the internal ecosystem vision to shape structure, but keep public copy buyer-friendly.

- Use the current offer files as source material for services copy.
- Keep the site maintainable and easy to update as offers evolve.
- Connect future lead capture to the lead intake and qualification workflow.
