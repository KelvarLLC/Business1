# Weekly Operator Review Template

## Purpose

Give Kelvar LLC a repeatable weekly review cadence for the Automation Agency Operator. This review connects pipeline, outreach, website launch readiness, intake status, proposals, delivery readiness, proof capture, KPIs, blockers, and next moves.

Use this after meaningful activity or at the end of each operating week. It is an internal operating record, not a client-facing report.

## When To Create A Weekly Review File

Create a dated review under:

```text
agency-ops/ops/reviews/YYYY-MM-DD-weekly-operator-review.md
```

Create one when:

- the first outreach batch is drafted or sent,
- a reply arrives,
- website build or preview evidence changes,
- Stage 0 intake QA is completed,
- a Stage 1 intake decision is made,
- a proposal/delivery event occurs,
- or a week ends with enough activity to summarize.
